# 📘 Ingeniería de Software 1 - UNLP (2026)

Bienvenido/a a este repositorio 📂 donde comparto **resoluciones, apuntes, prácticas y materiales** de la materia *Ingeniería de Software 1* (Facultad de Informática – UNLP).  
Este repositorio es parte de mi portafolio académico y está pensado para mostrar mi trabajo, mis avances y mis conocimientos reflejados en ejercicios reales de la materia.

- 🕸️ **Redes de Petri**
